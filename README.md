# Natural Language Processing Laboratory

Python implementations of core Natural Language Processing techniques including preprocessing, statistical language models, syntax analysis, semantic analysis, and practical NLP applications.

---

## Technologies

- Python
- NLTK
- SpaCy
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- WordCloud

---

## NLP Concepts Implemented

### Text Preprocessing
- Tokenization
- Stopword Removal
- Stemming
- Lemmatization

### Morphological Processing
- Inflectional Morphology
- Derivational Morphology
- Finite State Transducers

### Language Models
- N-gram Models
- Bigram
- Trigram
- Perplexity
- Laplace Smoothing
- Good-Turing Smoothing

### Syntax Processing
- Part-of-Speech Tagging
- Hidden Markov Model
- Viterbi Algorithm
- Chunking
- Parsing

### Semantic Processing
- WordNet
- Lexical Semantics
- Word Sense Disambiguation
- Lesk Algorithm

### NLP Applications
- Sentiment Analysis
- Spell Checker
- Plagiarism Detection
- Text Similarity
- Word Cloud Generation
