# Experiment 05 — Part-of-Speech Tagging using Hidden Markov Model (HMM)

## Overview

Part-of-Speech (POS) tagging is an important task in Natural Language Processing (NLP) that assigns grammatical categories to words in a sentence. These categories include nouns, verbs, adjectives, adverbs, and other parts of speech.

In this experiment, POS tagging is implemented using a **Hidden Markov Model (HMM)** approach. The HMM model is a statistical method that predicts the most likely sequence of POS tags based on probabilities learned from training data.

The experiment demonstrates how HMM can be used to automatically tag words in a sentence with their corresponding grammatical labels.

---

## Theory

Part-of-Speech tagging is the process of assigning grammatical labels to words in a sentence. For example:

Sentence:
The cat sits on the mat


POS Tagged Output:

| Word | POS Tag |
|-----|--------|
| The | DT |
| cat | NN |
| sits | VBZ |
| on | IN |
| the | DT |
| mat | NN |

Hidden Markov Model (HMM) is a probabilistic model used for sequence prediction problems. It assumes that:

1. The current state depends on the previous state.
2. The observed word depends on the hidden state (POS tag).

HMM uses two main probabilities:

- **Transition Probability:** Probability of moving from one POS tag to another.
- **Emission Probability:** Probability of a word being generated by a particular POS tag.

Using these probabilities, the model predicts the most likely POS tag sequence for a sentence.
